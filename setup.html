Set up your archive
One-time wizard · runs over the local server
  1. Source
  2. Group
  3. People
  4. Finish

1 · Point at your export

From your unzipped Twitter/X archive, pick all three pieces of your group export. Click Browse… for each (native pickers on Windows), or paste a path. All three are required.

Browse… open any file inside the media folder, we will use its folder.

✓ Archive already built. To change the source files, use Start over at the bottom of the page, then build again.

2 · Name the group optional

Restore the group's real name and photo. The photo becomes the sidebar brand mark. You can change these later in Settings.

💬

3 · Name everyone optional

The export only has numeric IDs. Use each person's sample messages & shared media (Twitter/X links excluded — they don't help) to recognize them. Add a name, a photo, and mark which one is you.

4 · Finish

Save your names, photos and "you" marker, then open the archive.

Erase all setup data?

This deletes everything the wizard created in personal_data/ — the built archive, names, photos, and the copied media & source files — so you can start setup from a clean slate. This cannot be undone.
